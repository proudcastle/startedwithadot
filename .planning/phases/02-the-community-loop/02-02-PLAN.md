<objective>
Add voting with optimistic UI and status tab filtering to the proposal feed. After this plan, users can toggle-vote on proposals with instant visual feedback, see their own votes highlighted, and filter the feed by proposal status using shareable URL-based tabs.

Purpose: Voting is the core engagement mechanic -- it determines which proposals rise to the top and get the admin's attention. Status tabs let users focus on the proposals they care about.
Output: Interactive vote buttons on every proposal card and a tab bar for status filtering.
</objective>
